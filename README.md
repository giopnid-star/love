# Valentine Site

Готовый статический сайт для публикации на Render через GitHub.

## Что уже сделано

- Инициализирован Git-репозиторий (`main`)
- Создан первый коммит
- Добавлен `render.yaml` для Static Site

## Осталось сделать (1-2 минуты)

1. Создай пустой репозиторий на GitHub (например `valentine-site`).
2. В терминале в этой папке выполни:

```bash
git remote add origin https://github.com/<your-username>/<your-repo>.git
git push -u origin main
```

3. На Render:
   - `New +` → `Static Site`
   - Подключи GitHub-репозиторий
   - Build Command: оставить пустым
   - Publish Directory: `.`
   - Нажми `Create Static Site`

После деплоя Render выдаст публичную ссылку на сайт.
